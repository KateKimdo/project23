Nutrition Information Portal - Designer, Developer

For this project, I created an interactive nutrition information platform using React, which allows users to search for the nutritional content of foods by typing in a search bar. Through the integration of the FoodData Central API, nutritional data is retrieved dynamically based on user input. The project combines custom HTML and CSS for structure and design, while the interactivity is driven by JavaScript and AJAX calls. By implementing a responsive design, I ensured that the site functions smoothly across different devices, and all the content and interactions are updated efficiently through React's state management system.
